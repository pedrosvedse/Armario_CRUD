# Armários Backend

## Dependências
- Node.js 18+
- npm

## Instalação
cd backend
npm install

## Execução
# Produção
npm start

# Desenvolvimento
npm run dev

## Rotas
| Método | Rota           | Descrição         |
|--------|----------------|-------------------|
| GET    | /armarios      | Listar todos      |
| GET    | /armarios/:id  | Buscar por ID     |
| POST   | /armarios      | Cadastrar         |
| PUT    | /armarios/:id  | Atualizar         |
| DELETE | /armarios/:id  | Excluir           |

## Tecnologias
- Node.js
- Express
- Cors
